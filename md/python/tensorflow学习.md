# tensorflow学习

本地python 使用 \Launcher 进行管理的

## numpy 安装 （Lib/site-packages 中有了，没尝试了）

    - py -3.13 -m pip install numpy

## tensorflow安装

- 安装CPU版本
  - py -3.13 -m pip install --upgrade tensorflow
