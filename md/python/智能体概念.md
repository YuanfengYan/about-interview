# 智能体（Agent）

## 智能体概念

- 编排型智能体
    一、知识库  
    二、 插件  
    三、 工作流  

- 代理型智能体 (OpenAI:Operator 、 智普：auto GLM 、Manus) 

## 开发智能体原则

1. 明确界限
   - 智能体要能做什么，你想让他做出什么样的具体结果 
2. 要设计容错和回滚机制
   - 去处理哪些AI肯定会犯的，20%到40%的错误案例
3. 解决钱的问题
    - 算好每次交互的成本是多少
    - 成本不会随着使用量增加而变化
    - 设计无状态模式要优于有状态模式
4. 优先考虑可靠性而不是自主性
5. AI的分工
